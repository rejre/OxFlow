【使用指南】
=============================================================================

【对于用户】
1. 阅读 QUICK_START.md (2分钟)
2. 阅读 DOCUMENTATION.md (深入了解)
3. 按照安装步骤操作

【对于开发者】
1. 克隆项目: git clone https://github.com/rejre/OxFlow.git
2. 安装依赖: make install-dev
3. 运行检查: make quality
4. 阅读 CONTRIBUTING.md 了解贡献流程

【常用命令】
make install          # 安装生产依赖
make install-dev      # 安装开发依赖
make test             # 运行测试
make lint             # 代码检查
make format           # 代码格式化
make quality          # 全面检查 (必做)
make run              # 运行程序
make clean            # 清理文件
